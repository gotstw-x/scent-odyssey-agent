# Odyssey of Scent | 选香之旅

A beginner-friendly, agentic perfume discovery experience (MSIS 549 HW2). Users “fish” vibe keywords, tune preferences with elegant sliders, and receive explainable perfume recommendations grounded in a seeded evidence dataset (official-style + reddit-style keywords). Community reviews feed back into the system for future improvement.
